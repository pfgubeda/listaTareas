# AppToDo
## Descripción
App Lista de tareas implementada con Angular. Visualizar, añadir, editar, completar y eliminar tareas.

## Características
- [x] Ver lista tareas
- [x] Añadir tarea
- [x] Editar tarea
- [x] Completar tarea
- [x] Eliminar tarea
- [x] Login page (sin implementación de backend)
- [x] Register page
- [x] ¿forgot password?
- [x] BackEnd Auth
- [x] LogIn con Google
- [x] Tareas Compartidas entre todos los users
- [ ] Log Out
